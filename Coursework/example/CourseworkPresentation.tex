\begin{frame}
    \frametitle{Основные понятия  \slnumber}
    \Large
   Наибольший общий делитель и наименьшее общее кратное:
    \begin{enumerate}
        \item Наибольший общий делитель целых $a$, $b$ "--- это такой их общий делитель, который делится на любой общий делитель этих чисел.        
       
        \item  Наименьшее общее кратное целых $a$, $b$ --- это такое наименьшее натуральное число, которое делится на $a$ и $b$.
    \end{enumerate}
\end{frame}

\begin{frame}
    \frametitle{Алгоритмы для нахождения наибольшего общего делителя  \slnumber}
    \Large
    Для вычисления $\nod$ двух натуральных чисел существуют следующие алгоритмы:
    \begin{enumerate}
         \item Античный алгоритм Евклида (через разности); 
         \item Алгоритм Евклида (через остатки);
         \item Нахождение $\nod$ методом перебора;
         \item Бинарный алгоритм Стейна.
    \end{enumerate}
    $\nok$ можно вычислить при помощи следующей формулы 
    \[\nok=\frac{a  b}{\nod(a,b)}.\]
\end{frame}

    \frametitle{Результаты курсовой работы  \slnumber}
    \large
    Эксперименты показали:
    \begin{enumerate}
        \item на первом, третьем и четвертом тесте алгоритм Евклида показывает лучшее время относительно других алгоритмов;
        \item на втором тесте время работы алгоритма Евклида равно времени выполнения бинарного алгоритма
        Стейна и античного алгоритма Евклида.
    \end{enumerate}   
    Из этого следует, что алгоритм Евклида является наиболее эффективным из
    рассмотренных.
